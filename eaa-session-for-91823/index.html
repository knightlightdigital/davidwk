EAA Session for 9/18/23

David K September 19, 2023

I was able to catch the Triangulum Galaxy (M33, pictured above) and Saturn last night. Both images are pretty rough but I was happy with the Saturn result. This was all done on the 80mm refractor. I'm going to attempt Jupiter and Saturn tonight with my Celestron 8se on the EAA mount and see if I can get better results. I'm also going to give it a try with the atmospheric dispersion corrector and see if I can remove some of the fringing.
